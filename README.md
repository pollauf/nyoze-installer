# Nyoze Installer

CLI tool for creating new Nyoze projects.

## Installation

```bash
composer global require pollauf/nyoze-installer
```

Make sure Composer's global `vendor/bin` directory is in your system PATH.

## Usage

```bash
nyoze new my-project
cd my-project
nyoze serve
```
